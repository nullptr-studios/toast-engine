Detailed Description

Interface for objects that support JSON serialization.

ISerializable defines the contract for objects that can save their state to JSON and restore it from JSON. This is used for scene files, prefabs, and runtime state persistence.

Implementing Serialization:
class MyActor : public Actor {
public:
json_t Save() const override {
auto j = Actor::Save(); // Call parent
j["health"] = m_health;
j["score"] = m_score;
return j;
}
void Load(json_t j, bool force_create) override {
Actor::Load(j, force_create); // Call parent
m_health = j.value("health", 100.0f);
m_score = j.value("score", 0);
}
private:
float m_health = 100.0f;
int m_score = 0;
};
