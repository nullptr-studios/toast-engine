Detailed Description

Struct used to search for actors in scenes.

To use this ActorLookup we make use of the Designated Initializers added in C++ 20. This allows us to just have one function that handles all the different searching syntaxes (by id or by name) instead of having multiple functions overridden. The usage of this is very simple:

// This will all return the same
World::GetActor({.sceneName = "testScene", .actorName = "Player"});
World::GetActor({.sceneID = 0, .actorName = "Player"});
// If you don't enter a scene, the system will assume you are referring to the main scene
World::GetActor({.actorName = "Player"});
// If you enter both a name or an ID the system will use the ID since it's more performant
World::GetActor({.sceneID = 0, .sceneName = "testScene", .actorName = "Player"});
