Detailed Description

template<ActionValues Value>
struct input::Action< Value >

An Input Action is a struct that contains all the information about any action the player can perform. They are used as the parameter for the action callback and contain all the information we might need about the button press.

To use them just get the information you need directly from the struct. To get the actual value see GetValue(glm::vec2* variable)
