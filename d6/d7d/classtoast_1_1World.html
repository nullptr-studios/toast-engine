Detailed Description

Class that contains all the loaded scenes on the game.

The World class is a Singleton that contains all the currently loaded scenes on the engine and is in charge of managing and ticking them. The client interacts with this class to load and unload scenes and to enable or disable them. It is also used to search for game objects between all the scenes
