Detailed Description

Rendering layer for HTML/CSS/JS based UI using Ultralight.

This layer manages Ultralight Views for rendering web-based UI elements. It integrates with the Toast Engine's rendering pipeline and handles GPU-accelerated rendering of web content.

The HUD renders to its own framebuffer which can be retrieved via GetFramebuffer() for compositing by the main renderer.

Usage Example:
auto hudLayer = std::make_unique<HUDLayer>(window, 1920, 1080);
hudLayer->OnAttach();
hudLayer->LoadURL("file:///assets/ui/main_menu.html");
// In game loop:
hudLayer->OnTick();
hudLayer->OnRender();
// Get the framebuffer texture for compositing
GLuint hudTexture = hudLayer->GetFramebuffer()->GetColorTexture(0);
