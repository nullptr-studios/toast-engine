 CActorThe simplest Object that exists on the game world An actor is the most basic game object that can be spawned into the game world. It is the base for all the things that exist in game. The main difference with the Object class is the ability to have both child Actors and Components. The object philosophy follows a tree with top-left traversal where the children are managed by their respective parents. Each actor class is responsible for the ownership, creation, destruction and ticking of all the children it has
