Loads a resource of the specified type.

If the resource is already cached, returns the cached version. Otherwise, loads from disk (or pkg) and caches it.

Template Parameters
RResource type (must inherit from IResource).
ArgsAdditional constructor argument types.
Parameters
pathPath to the resource file (relative to assets/).
argsAdditional arguments passed to the resource constructor.
Returns
Shared pointer to the loaded resource.
Example:
