Detailed Description

Abstract base class for all renderer implementations.

This class defines the interface that all renderers must implement and provides common functionality for camera management, scene management, and matrix operations. It uses a singleton pattern to allow global access to the active renderer instance.

Note
Derived classes should implement specific rendering pipelines (forward, deferred, etc.)
