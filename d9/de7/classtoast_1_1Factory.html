Detailed Description

Handles all the logic for creating and destroying Actors.

This class was in charge of the creation of any kind of Object and assign it a unique ID

Note
This class shouldn't be used by the end user but by the implementation of the Actor tree structure or any other that works as some sort of middleware

All the actors that exist in the Game World should be created using the factory class, as they need to perform some additional logic aside from just allocating the memory

Use the factory to create or destroy any Actor
