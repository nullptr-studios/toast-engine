# Toast Engine

Toast Engine is an open-source game engine created by nullptr* Studios to
develop their games during their scholarship on Digipen Institute of
Technology.

**Requirements**

**Windows**
- Visual Studio 2022 or higher
- CMake 3.15 or higher
- Xmake

**Linux**
- GCC or Clang
- CMake 3.15 or higher
- Xmake

**Instructions**
1. `xmake f -m debug` to enter debug mode instead of release
2. `xmake run` to run the editor
    If you want to run the test app just do `xmake run toast.test`
